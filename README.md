jQuery-ribbon
=============

jQuery ribbon - animated ribbon

Usage 
=============
		$('.grid a').ribbon({
			speedIn: 200, // speed to animate in
			speedOut: 200, // speed to animate out
			title: 'alt', // derive text from first 'alt' or 'text'
			color: 'img:first' // element to look at for class name for color
		});
